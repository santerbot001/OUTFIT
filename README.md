# OUTFIT — Customer Frontend
Premium fashion e-commerce storefront. **Style Beyond Trends.**

## Stack
React 18 · Vite 5 · React Router 6 · Framer Motion · Lenis (smooth scroll) · Lucide icons

## Run
\`\`\`bash
npm install
npm run dev
\`\`\`
Build: \`npm run build\` · Preview: \`npm run preview\`

## Design tokens
Pure white #FFFFFF bg · black #0A0A0A accent · Poppins headings + Inter body · 12–16px radius. See \`src/styles/tokens.css\`.
